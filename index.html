Minha Biblioteca: Uma Webpage Personalizada

Esse projeto é uma página web focada em receitas de brownies que desenvolvi usando HTML e CSS. A página foi criada para compartilhar receitas deliciosas e fáceis, apresentando passo a passo o preparo dos brownies e dicas para um resultado perfeito..

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

A estrutura do site é feita com HTML, que organiza o conteúdo em seções como o título, uma breve introdução, a lista de ingredientes e o modo de preparo. Cada receita é apresentada em um contêiner (.receita) que exibe os ingredientes necessários, uma foto do brownie pronto e instruções detalhadas..

O CSS personaliza a aparência da página, aplicando uma paleta de cores aconchegante e tipografias que deixam o site visualmente agradável. Fontes externas do Google Fonts são usadas para dar um toque especial, e o layout é responsivo para se adaptar a diferentes dispositivos..
